LIRI

A Language Interpretation & Recognition Interface; LIRI takes user input from a series of commands. Interfaces with various APIs and uses multiple Node packagaes. One of the best features is its scalability - it can be greatly (and easily!) expanded!


    Technologies:
  • Node.js
  • Inquirer, Axios, Spotify, Moment.js & FS
  • Colors node package for a fun interface
